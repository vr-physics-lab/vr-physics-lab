# 🔒 Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | ✅ Yes |
| < v1.0  | ⚠️ Best effort |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue**.

Instead, email us at: **security@vrphysicslab.org**

We will respond within **72 hours** and provide a fix or mitigation plan.

Thank you for helping keep VR Physics Lab and its users safe.
