# ✅ OSS Program Application Checklist

This checklist helps ensure VR Physics Lab qualifies for open-source programs like **Claude for OSS** and **Codex for OSS**.

---

## 📌 Repository Health

- [ ] Public GitHub repository
- [ ] Clear open-source license (MIT) in root
- [ ] Detailed README with project description
- [ ] CONTRIBUTING.md with clear contribution guide
- [ ] CODE_OF_CONDUCT.md
- [ ] SECURITY.md
- [ ] Active issue tracker with labeled issues
- [ ] GitHub Discussions enabled
- [ ] At least 1 release or pre-release
- [ ] CI/CD pipeline configured (GitHub Actions)

## 📄 Documentation

- [ ] Setup/installation guide
- [ ] Architecture overview
- [ ] API/SDK documentation
- [ ] Roadmap
- [ ] Changelog

## 🤝 Community

- [ ] `good first issue` labels on beginner-friendly issues
- [ ] Responded to issues within 72h
- [ ] At least 1 external contributor (PR merged)
- [ ] GitHub Discussions with active threads
- [ ] Project board showing active work

## 🏆 Project Credibility

- [ ] Government grant documentation referenced
- [ ] Clear educational mission statement
- [ ] Institutional affiliation mentioned
- [ ] Real-world deployment plan described

## 🤖 AI Integration Justification (for Claude/Codex OSS)

- [ ] Describe how Claude/Codex would be used in the project
  - Example: AI-powered physics tutoring inside VR
  - Example: Automated experiment hint generation
  - Example: Code generation for new experiment modules
- [ ] Show that the project is actively developed
- [ ] Estimate usage (API calls, features to build)

---

## 📝 Application Talking Points

When applying to Claude for OSS or Codex for OSS, highlight:

1. **Government-funded** — Official "Bo'lajak olim" grant winner, Republic of Uzbekistan
2. **Educational impact** — Targeting schools and universities in Central Asia
3. **Open-source ecosystem** — Not just one repo, but an SDK + assets + docs ecosystem
4. **Active development** — Grant implementation stage, real deployment planned
5. **AI use case** — AI tutoring layer to provide real-time physics explanations in VR
6. **Community** — Open to international contributors, educators, VR developers
7. **Long-term vision** — Expanding to full multi-subject STEM platform
