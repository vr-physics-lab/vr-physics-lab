# 🗺️ VR Physics Lab — Roadmap

This roadmap outlines our planned milestones from initial prototype to a full multi-subject STEM platform.

> 📌 This project is funded by the "Bo'lajak olim" government innovation grant (Uzbekistan).

---

## 🔷 v0.1 — Foundation (Q1 2025)

**Goal:** Establish the core VR framework and first working experiment.

- [ ] Unity project setup with Meta XR SDK
- [ ] OpenXR + XR Interaction Toolkit integration
- [ ] VR locomotion system (teleport + smooth)
- [ ] Hand tracking & controller support
- [ ] Newton's Laws interactive experiment
  - [ ] Force application demo (push objects)
  - [ ] Mass vs. acceleration visualization
  - [ ] Action-reaction demonstration
- [ ] Basic in-VR UI (menu, pause, info panels)
- [ ] Git LFS setup for large assets
- [ ] Initial CI/CD pipeline

---

## 🔷 v0.2 — Gravity & Free Fall (Q2 2025)

**Goal:** Add gravity experiments with multi-planet support.

- [ ] Configurable gravity environments (Earth, Moon, Mars, Jupiter)
- [ ] Free fall measurement tools
- [ ] Projectile motion sandbox
- [ ] Real-time trajectory visualization
- [ ] Data recording in VR

---

## 🔷 v0.3 — Energy Conservation (Q3 2025)

**Goal:** Visualize energy transfer in real-time.

- [ ] Kinetic ↔ Potential energy visualization
- [ ] Pendulum experiment with energy display
- [ ] Roller-coaster energy demo
- [ ] Energy graphs overlay in VR
- [ ] Thermal energy introduction

---

## 🔷 v0.4 — Collisions Sandbox (Q3 2025)

**Goal:** Interactive collision experiments.

- [ ] Elastic collision simulator
- [ ] Inelastic collision simulator
- [ ] Adjustable mass and velocity controls
- [ ] Momentum conservation visualization
- [ ] Slow-motion replay system

---

## 🔷 v1.0 — Full Prototype (Q4 2025)

**Goal:** Complete, deployable prototype for schools.

- [ ] All physics modules integrated
- [ ] Teacher dashboard (non-VR)
- [ ] Student progress tracking
- [ ] Multi-language support (English, Uzbek, Russian)
- [ ] Comprehensive documentation
- [ ] School pilot program (Uzbekistan)
- [ ] Performance optimization for Quest 3
- [ ] Accessibility features

---

## 🔷 v2.0 — STEM Platform (2026+)

**Goal:** Expand to a full multi-subject STEM platform.

- [ ] Chemistry Lab module
- [ ] Engineering experiments
- [ ] Technical Education module
- [ ] Multiplayer classroom mode
- [ ] LMS integration (Moodle, Google Classroom)
- [ ] Content creation tools for teachers
- [ ] Mobile/PC companion app

---

## 💡 Long-term Vision

> Build a scalable, open-source STEM education ecosystem accessible to every school — regardless of location or resources.
